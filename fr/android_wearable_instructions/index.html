Рour ces appareils les instructions sont identiques car ils sont directement connectés à Google Fit

Pour connecter Zepp Life à l'application :

  • Connectez votre bracelet ou votre smartwatch à l'application Zepp Life
  • Commencez n'importe quelle séance d'entraînement. Avant de commencer une séance d'entraînement, l'application vous invite à connecter votre bracelet pour suivre les données

Si vous n'êtes pas invité à le faire, vous devriez procéder ainsi :

  1. Ouvrez l'application FitnessOnline
  2. Appuyez sur "Plus" dans le coin inférieur droit
  3. Appuyez sur "Paramètres"
  4. Appuyez sur "L'èchange de donnèes"
  5. Appuyez sur l'icône de l'application
  6. Appuyez sur "Connecter"

Si le bracelet était précédemment connecté mais qu'il ne transfère pas de données, vous devriez procéder ainsi :

  1. Vérifiez que le bracelet est connecté à l'application Zepp Life. Pour ce faire, vous devez ouvrir l'application Zepp Life, aller dans "Profil" et vérifier que le bracelet est connecté sous "Mes appareils"
  2. Allez dans l'application FitnessOnline
  3. Appuyez sur "Plus" dans le coin inférieur droit
  4. Appuyez sur "Paramètres"
  5. Appuyez sur "L'èchange de donnèes"
  6. Appuyez sur l'icône de l'appli
  7. Appuyez sur "Repèter la connexion"

Si votre bracelet est connecté mais que seules quelques calories ont été comptées :

Cela se produit lorsque le bracelet calcule lui-même le nombre de calories brûlées alors que le système d'entraînement n'est pas activé.

Pour afficher un nombre plus précis de calories brûlées, lancez une séance d'entraînement sur le bracelet.

Pour connecter Google Fit à l’application :

  • Commencez n'importe quelle séance d'entraînement. Avant de commencer une séance d'entraînement, vous serez invité à vous associer si l'application Google Fit est déjà installée.

Si vous n’êtes pas invité à le faire, vous devriez procéder ainsi :

  1. Allez sur l'application FitnessOnline
  2. Appuyez sur "Plus" dans le coin inférieur droit
  3. Appuyez sur "Paramètres"
  4. Appuyez sur "L'èchange de donnèes"
  5. Appuyez sur l'icône Google Fit
  6. Appuyez sur "Se connecter avec Google"

Si les données n'apparaissent pas dans le rapport, vous devriez procéder ainsi :

  1. Vérifiez si les données sont transférées à Google Fit

  2. Vérifiez la connexion entre Google Fit et l'application. Pour ce faire, effectuez les étapes suivantes dans l'application FitnessOnline :

    1. Appuyez sur "Plus" dans le coin inférieur droit
    2. Appuyez sur "Paramètres"
    3. Appuyez sur "L'èchange de donnèes"
    4. Appuyez sur l'icône Google Fit

Pour connecter Fitbit, LG, Polar à l’application :

Afin de connecter votre application utilisée pour suivre votre activité avec l'application FitnessOnline, vous devez synchroniser votre application avec Google Fit.

Vous devriez procéder ainsi :

  1. Installez l'application Google Fit
  2. Créez un compte
  3. Ouvrez l'application que vous utilisez pour suivre votre activité
  4. Allez dans les paramètres
  5. Trouvez la section permettant de connecter d'autres apps. Elle s'appelle normalement "Connecter d'autres services", "Apps et appareils", "Gérer les connexions", etc.
  6. Suivez les instructions pour connecter l'application à Google Fit

Ensuite, vous devez synchroniser Google Fit et l'application FitnessOnline en procédant comme suit :

  • Commencez n'importe quelle séance d'entraînement. Avant de commencer une séance d'entraînement, vous serez invité à associer

Si vous n’êtes pas invité à le faire, vous devriez procéder ainsi :

  1. Allez sur l'application FitnessOnline
  2. Appuyez sur "Plus" dans le coin inférieur droit
  3. Appuyez sur "Paramètres"
  4. Appuyez sur "L'èchange de donnèes"
  5. Appuyez sur l'icône Google Fit
  6. Appuyez sur "Se connecter avec Google"

Si les données n’apparaissent pas dans le rapport, vous devriez procéder ainsi :

  1. Vérifiez si les données sont transférées à Google Fit
  2. Vérifiez que Google Fit et l'application sont connectés en allant dans Paramètres — L'èchange de donnèes — Google Fit

Afin de connecter votre application utilisée pour suivre votre activité avec Google Fit, vous devez :

  1. Ouvrez l'application Google Fit
  2. Appuyez sur "Profil" — Appuyez sur "Paramètres"
  3. Appuyez sur "Gérer les applications connectées"
  4. Pour ouvrir la liste de toutes les applis connectées, appuyez sur la flèche vers le bas sous "Toutes les applis et tous les appareils"

Pour connecter Huawei Health à FitnessOnline, vous devez utiliser Google Fit

Pour ce faire, vous devrez utiliser l'application tierce Health Sync (une application payante). Si vous ne pouvez pas utiliser une application payante pour synchroniser les données, vous pouvez retourner à la fenêtre précédente et sélectionner la synchronisation avec Google Fit (le bracelet ne sera pas pris en compte).

Pour utiliser une application tierce, vous devez effectuer les opérations suivantes sur Play Market :

  1. Installez Google Fit
  2. Installez l'application Health Sync. Lancez l'appli et sélectionnez le Huawei Health Kit comme source principale à synchroniser et sélectionnez Google Fit comme direction de synchronisation
  3. Appuyez sur "Déclaration de consentement"

  4. Ensuite, vous devrez suivre les instructions de l'app Health Sync, à savoir :

    • Appuyez sur “Vérifier la connexion au compte Google”
    • Appuyez sur “Vérifier le suivi des activités dans Google Fit”
    • Installez Huawei AppGallery (copie des instructions du site official https://consumer.huawei.com/fr/support/content/fr-fr00696141/)
    • Installez HMS CORE (Pour ce faire, ouvrez AppGallery et entrez-y HMS CORE). AppGallery demandera la permission d'installer à partir de ce produit et la permission doit être accordée
    • Installez Huawei Health (ou mettez à jour)
    • Сonnectez-vous à votre compte Huawei
    • Cochez les cases nécessaires à la synchronisation de l'application Health Sync

  5. Pour connecter Google Fit à l'application:

    • Commencez n'importe quelle séance d'entraînement. Avant de commencer une séance d'entraînement, vous serez invité à vous associer si l'application Google Fit est déjà installée

Si vous n’êtes pas invité à le faire, vous devriez procéder ainsi :

  1. Allez sur l'application FitnessOnline
  2. Appuyez sur "Plus" dans le coin inférieur droit
  3. Appuyez sur "Paramètres"
  4. Appuyez sur "L'èchange de donnèes"
  5. Appuyez sur l'icône Google Fit
  6. Appuyez sur "Se connecter avec Google"

Si les données n’apparaissent pas dans le rapport, vous devriez procéder ainsi :

  1. Vérifiez si les données sont transférées à Google Fit
  2. Vérifiez que Google Fit et l'application sont connectés en allant dans Paramètres — L'èchange de donnèes — Google Fit
