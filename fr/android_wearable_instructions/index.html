Ces appareils ont une connexion directe à l'application Health Connect

Pour synchroniser Google Health Connect (disponible dans les versions 2.20.0 et supérieures de FitnessOnline) avec l'application, suivez ces étapes :

  1. Installez l'application Health Connect sur votre appareil.
  2. Après l'installation initiale de FitnessOnline sur votre appareil et après vous être connecté à votre compte, accordez l'accès pour lire les données dans Health Connect.

Si l'accès à la lecture des données dans Health Connect n'a pas été accordé après l'installation initiale de l'application :

  1. Appuyez sur “Plus” dans le coin inférieur droit.
  2. Appuyez sur “Paramètres”.
  3. Appuyez sur “Partage de données”.
  4. Appuyez sur l'icône Health Connect.
  5. Appuyez sur “Ouvrir les paramètres”.
  6. Accordez l'accès pour lire les données pour l'application FitnessOnline.

Si les données ne s'affichent pas sur l'écran du résumé d'entraînement, suivez ces étapes :

  1. Vérifiez si les données sont synchronisées avec Health Connect.

  2. Vérifiez l'état de la synchronisation entre Health Connect et FitnessOnline en exécutant les étapes suivantes dans l'application FitnessOnline :

    1. Appuyez sur “Plus” dans le coin inférieur droit.
    2. Appuyez sur “Paramètres”.
    3. Appuyez sur “Partage de données”.
    4. Appuyez sur l'icône Health Connect.

Pour synchroniser Zepp Life avec FitnessOnline :

  1. Connectez votre bracelet fitness ou montre intelligente à l'application Zepp Life.
  2. Lancez n'importe quelle séance d'entraînement dans FitnessOnline. Avant l'entraînement, l'application vous demandera de synchroniser votre bracelet fitness pour le suivi des données.

Si l'invite de synchronisation n'apparaît pas, suivez ces étapes :

  1. Ouvrez l'application FitnessOnline.
  2. Appuyez sur “Plus” dans le coin inférieur droit.
  3. Appuyez sur “Paramètres”.
  4. Appuyez sur “Partage de données”.
  5. Appuyez sur l'icône de l'application.
  6. Appuyez sur “Connecter”.

Si votre bracelet fitness est connecté mais que les données ne se synchronisent pas :

  1. Éteignez le Bluetooth puis rallumez-le.
  2. Vérifiez l'état de la synchronisation de votre bracelet fitness avec l'application Zepp Life. Pour ce faire, ouvrez l'application Zepp Life, allez à l'onglet “Profil” et dans la section “Mes appareils”, assurez-vous que votre bracelet fitness est connecté.
  3. Ouvrez l'application FitnessOnline.
  4. Appuyez sur “Plus” dans le coin inférieur droit.
  5. Appuyez sur “Paramètres”.
  6. Appuyez sur “Partage de données”.
  7. Appuyez sur l'icône de l'application.
  8. Appuyez sur “Reconnecter”.

Si votre bracelet fitness est connecté mais indique peu de calories sur l'écran de résumé d'entraînement :

Cela se produit parce que votre bracelet fitness calcule ce montant spécifique de calories brûlées.

Pour un affichage plus précis des calories brûlées, vous devriez commencer une séance d'entraînement système sur votre bracelet fitness.

Si l'invite de synchronisation n'apparaît pas, suivez ces étapes :

  1. Ouvrez l'application FitnessOnline.
  2. Appuyez sur “Plus” dans le coin inférieur droit.
  3. Appuyez sur “Paramètres”.
  4. Appuyez sur “Partage de données”.
  5. Appuyez sur l'icône pour Google Fit.
  6. Appuyez sur “Se connecter avec Google”.

Si les données ne s'affichent pas sur l'écran du résumé d'entraînement, suivez ces étapes :

  1. Vérifiez si les données sont synchronisées avec Google Fit.

  2. Vérifiez l'état de la synchronisation entre Google Fit et FitnessOnline en exécutant les étapes suivantes dans l'application FitnessOnline :

    1. Appuyez sur “Plus” dans le coin inférieur droit.
    2. Appuyez sur “Paramètres”.
    3. Appuyez sur “Partage de données”.
    4. Appuyez sur l'icône pour Google Fit.

Le transfert de données de Samsung Health, Garmin, Fitbit et Polar Flow vers FitnessOnline est facilité via Google Health Connect

Pour se connecter, suivez ces étapes :

  1. Installez l'application Health Connect et connectez-vous.
  2. Ouvrez l'application utilisée pour suivre votre activité.
  3. Accédez au menu des paramètres.
  4. Recherchez la section pour connecter des applications tierces. Elle est généralement étiquetée comme “Connecter d'autres services”, “Applications et appareils” ou “Gérer les connexions” ou quelque chose de similaire.
  5. Suivez les instructions pour connecter l'application à Health Connect.

Ensuite, synchronisez Health Connect avec l'application FitnessOnline comme suit :

  1. Dans l'application FitnessOnline, appuyez sur “Plus” dans le coin inférieur droit.
  2. Appuyez sur “Paramètres”.
  3. Appuyez sur “Partage de données”.
  4. Appuyez sur l'icône Health Connect.
  5. Appuyez sur “Ouvrir les paramètres”.
  6. Accordez l'accès pour lire les données pour l'application FitnessOnline.

Si les données ne s'affichent pas sur l'écran du résumé d'entraînement, suivez ces étapes :

  1. Vérifiez si les données de l'application utilisée pour suivre les activités sont synchronisées avec Health Connect.

  2. Vérifiez l'état de la synchronisation entre Health Connect et FitnessOnline en exécutant les étapes suivantes dans l'application FitnessOnline :

    1. Appuyez sur “Plus” dans le coin inférieur droit.
    2. Appuyez sur “Paramètres”.
    3. Appuyez sur “Partage de données”.
    4. Appuyez sur l'icône Health Connect.

Pour vérifier la connexion entre Health Connect et l'application utilisée pour suivre les activités, suivez ces étapes :

  1. Ouvrez l'application Health Connect.
  2. Dans la section “Autorisations et données”, appuyez sur “Autorisations des applications”.
  3. Assurez-vous que l'application que vous utilisez est répertoriée sous “Accès autorisé”.

Si votre bracelet fitness et Health Connect sont connectés, mais que des données d'activité incorrectes ou aucune donnée n'apparaissent sur l'écran du résumé d'entraînement :

Cela se produit parce que l'application de suivi des activités envoie des données spécifiques de calories, de nombre de pas et de fréquence cardiaque à Health Connect.

FitnessOnline affiche uniquement les données reçues de Health Connect.

Pour garantir l'affichage de données d'activité plus précises sur l'écran du résumé d'entraînement :

  1. Avant de commencer une séance d'entraînement dans FitnessOnline, commencez une séance d'entraînement système sur votre bracelet fitness.
  2. Avant de terminer la séance dans FitnessOnline, ouvrez l'application utilisée pour suivre les activités et actualisez la synchronisation entre votre bracelet fitness et l'appareil mobile.

Le transfert de données de l'application Huawei Health vers FitnessOnline est facilité via Google Health Connect et Health Sync (nécessite un abonnement payant)

Pour connecter, suivez ces étapes :

  1. Installez l'application Health Connect et connectez-vous.
  2. Installez l'application Health Sync. Ouvrez l'application et sélectionnez Huawei Health Kit comme source principale de synchronisation; pour la direction de synchronisation, choisissez Health Connect.
  3. Appuyez sur le bouton “Déclaration de consentement”.

  4. Ensuite, suivez les instructions dans l'application Health Sync :

    • Appuyez sur “Vérifier la connexion au compte Google”.
    • Appuyez sur “Vérifier le suivi des activités dans Health Connect”.
    • Installez Huawei AppGallery (consultez les instructions officielles : Support Huawei).
    • Installez HMS CORE depuis l'AppGallery (l'AppGallery nécessitera l'autorisation du système pour installer des applications tierces).
    • Installez ou mettez à jour Huawei Health.
    • Connectez-vous à votre compte Huawei.
    • Dans Health Sync, cochez les cases pour les types de données que vous souhaitez transférer à Health Connect.
  5. Assurez-vous que Health Sync est connecté avec succès à Huawei Health dans les paramètres de “Gestion de la confidentialité” de l'application Huawei Health.

Ensuite, synchronisez Health Connect avec l'application FitnessOnline comme suit :

  1. Dans l'application FitnessOnline, appuyez sur “Plus” dans le coin inférieur droit.
  2. Appuyez sur “Paramètres”.
  3. Appuyez sur “Partage de données”.
  4. Appuyez sur l'icône Health Connect.
  5. Appuyez sur “Ouvrir les paramètres”.
  6. Accordez l'accès pour lire les données pour l'application FitnessOnline.

Si les données ne s'affichent pas sur l'écran du résumé d'entraînement, suivez ces étapes :

  1. Vérifiez si les données de Health Sync se synchronisent avec Health Connect.

  2. Vérifiez l'état de la synchronisation entre Health Connect et FitnessOnline en exécutant les étapes suivantes dans l'application FitnessOnline :

    1. Appuyez sur “Plus” dans le coin inférieur droit.
    2. Appuyez sur “Paramètres”.
    3. Appuyez sur “Partage de données”.
    4. Appuyez sur l'icône Health Connect.

Pour vérifier la connexion entre Health Connect et Health Sync, suivez ces étapes :

  1. Ouvrez l'application Health Connect.
  2. Dans la section “Autorisations et données”, appuyez sur “Autorisations des applications”.
  3. Assurez-vous que Health Sync figure sous “Accès autorisé”.

Si votre bracelet fitness, Health Sync et Health Connect sont connectés, mais que des données d'activité incorrectes ou aucune donnée n'apparaissent sur l'écran du résumé d'entraînement :

Cela se produit parce que l'application de suivi des activités envoie des données spécifiques de calories, de nombre de pas et de fréquence cardiaque à Health Sync. Aussi, il se peut que Health Sync envoie des données à Health Connect avec un certain retard.

FitnessOnline affiche uniquement les données reçues de Health Connect.

Pour garantir l'affichage de données d'activité plus précises sur l'écran du résumé d'entraînement, suivez ces étapes :

  1. Avant de commencer une séance d'entraînement dans FitnessOnline, commencez une séance d'entraînement système sur votre bracelet fitness.
  2. Avant de terminer la séance dans FitnessOnline, ouvrez l'application utilisée pour suivre les activités et actualisez la synchronisation entre votre bracelet fitness et l'appareil mobile.
  3. Ouvrez l'application Health Sync et actualisez la synchronisation avec Health Connect.
  4. Retournez à l'application FitnessOnline et terminez la séance.
