对这些设备来说使用指南是完全相同的因为它们都是直接连接到谷歌运动

为了用app连接zepp life

  • 连接你的手环或者智能手表至zepp life 应用
  • 开始任何锻炼。在开始锻炼前，app会提示你连接你的手环来追踪数据

如果你没有被提示你应该：

  1. 找到线上健康应用
  2. 轻触右下角的更多
  3. 轻触设置
  4. 轻触数据交换
  5. 轻触应用图标
  6. 轻触连接

如果手环之前已连接了但是现在却不转移数据了你应该：

  1. 检查手环是否连接到了zepp life 软件上，你需要打开zepp life软件，选择资料，并确保手环在我的设备中已被连接
  2. 去线上健康app
  3. 轻触右下角的更多按钮
  4. 轻触设置
  5. 轻触数据交换
  6. 轻触应用的图标
  7. 轻触重复连接

如果你的手环已连接了但是只有很少的卡路里被算入：

这种情况在手环自己记录燃烧的卡路里而系统锻炼没有被打开时会发生。为了显示更准确的燃烧的卡路里数量，在手环上开启一次锻炼。

用app连接谷歌运动：

  • 开始任何锻炼。在开始锻炼前如果你已安装了谷歌运动app，你会被邀请配对

如果你没有被提示你应该：

  1. 找到线上健康应用
  2. 轻触右下角的更多
  3. 轻触设置
  4. 轻触数据交换
  5. 轻触谷歌运动图标
  6. 轻触用谷歌账号登录

如果数据没有在报告中显示你应该：

  1. 检查数据是否被转移到了谷歌运动上。

  2. 检查谷歌运动和app的连接，在线上健康app中完成如下步骤：

    1. 轻触右下角的更多
    2. 轻触设置
    3. 轻触数据交换
    4. 轻触谷歌运动图标

为了用app连接Fitbit, LG, Polar

为了连接你的app并用谷歌运动追踪你的活动你需要用谷歌运动来同步你的app

你应该：

  1. 安装谷歌运动app
  2. 创建一个账户
  3. 打开你用于追踪活动的app
  4. 来到设置
  5. 找到连接其他app的部分，它通常被称作“连接其他服务”“应用和设备”“管理连接”等待
  6. 遵照用谷歌运动连接app的指示

接下来你需要通过如下操作来同步谷歌运动和线上健康app

  • 开始任何锻炼。在开始锻炼前你会被邀请配对

如果你没有被提示你应该：

  1. 找到线上健康应用
  2. 轻触右下角的更多
  3. 轻触设置
  4. 轻触数据交换
  5. 轻触谷歌运动图标
  6. 轻触用谷歌账号登录

如果数据没有在报告中显示你应该：

  1. 检查数据是否被转移到了谷歌运动上。
  2. 检查谷歌运动和应用是否连接，找到设置--数据交换--谷歌运动

为了连接你的app并用谷歌运动追踪你的活动你应该：

  1. 打开谷歌运动app
  2. 轻触资料 -- 轻触设置
  3. 轻触管理已连接的应用
  4. 打开全部已连接应用的列表，按下在“所有应用和设备”下的箭头

如果你的手环已连接了但是只有很少的卡路里被算入：

这种情况在手环自己记录燃烧的卡路里而系统锻炼没有被打开时会发生。为了显示更准确的燃烧的卡路里数量，在手环上开启一次锻炼。

为了用线上健康来连接华为健康，你需要使用谷歌运动

为了完成这个目标，你需要使用第三方健康同步软件（付费使用）。如果你不使用付费使用的软件来同步数据，你可以回到之前的窗口，选择用谷歌运动同步（手环数据不会被考虑在内）

为了使用第三方应用，你需要在play market完成如下指示：

  1. 安装谷歌运动
  2. 安装健康同步应用，启动软件，选择Garmin作为主要来源来同步并选择谷歌运动作为同步方向
  3. 轻触同意书

  4. 然后你需要遵照健康同步app上的指示，也就是：

    • 轻触“检查谷歌账户连接”
    • 轻触“检查谷歌运动中的活动追踪”
    • 安装华为应用商城（来自官网的指示副本： https://consumer.huawei.com/my/mobileservices/appgallery/android-installation)
    • 安装HTMS CORE（为了打开华为应用商城并在这里进入HTMS CORE）。应用商城需要请求从该产品的安装许可，许可必须被授予
    • 安装华为健康（或者更新）
    • 认证或者登录华为账户
    • 检查需要同步的健康同步软件的选框

  5. 连接谷歌运动和app

    • 开始任何锻炼。开始锻炼前，如果已安装了谷歌运动app，你会被邀请配对。

如果你没有被提示你应该：

  1. 找到线上健康应用
  2. 轻触右下角的更多
  3. 轻触设置
  4. 轻触数据交换
